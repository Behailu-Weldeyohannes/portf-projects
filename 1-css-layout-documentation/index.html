Introduction

CSS provides many tools and properties that we can use to position elements on a website.The Flexible Box Module is a one-dimensional layout model, and method that is designed to offer space distribution between items in an interface and powerful alignment capabilities. On the other hand, CSS Grid is a set of css properties for building 2-dimensional layouts. Both are designed to write fewer and cleaner html and css codes.This documentation gives an outline of the main features of flexbox.

What is the Flexbox

Flexbox is a tool that is used in CSS to easily move items around within their containers. This is essential for the layout and organization of your website. Flexbox is tool to build layouts. Flexbox deals with layout in one dimension at a time — either as a row or as a column. In a nutshell, Flexbox is a set of related css properties for building one dimensional layouts.

.container{ display: flex;}

Flexbox automatically align items to one another inside a parent container. Flexbox provides a cross browser-compatible method to implement layouts and makes it easier to add responsive design to your application, without you having to explicitly deal with positioning the components of your page.

The Two Axes

There are two axis, the main axis and the cross axis. The main axis is defined by the flex-direction property, and the cross axis runs perpendicular to it. Everything we do with flexbox refers back to these axes. By default, the main axis refers to the x-axis or the horizontal axis. Likewise, the cross axis refers to the y-axis or the vertical axis.

the Main Axis

The main axis is defined by flex-direction.The flex-direction CSS property specifies how flex items are placed in the flex container - either vertically or horizontally. This property also determines whether those flex items appear in order or in reverse order.

Flex-direction has four possible values:

  • row
  • row-reverse
  • column
  • column-reverse

.container {
display: flex;
flex-direction: column;
}

The Cross Axis

The cross axis runs perpendicular to the main axis.For ecample, if the flex-direction (main axis) is set to row or row-reverse the cross axis runs down the columns. If the main axis is column or column-reverse then the cross axis runs along the rows.

Flex-direction has four possible values:

  • row
  • row-reverse
  • column
  • column-reverse
The flex Container: Properties for the Parent

There are two important components to a flexbox layout: flex containers and flex items. Flex container is an area of a document laid out using flexbox. The flex container is the main element that flex items are placed inside.

The flex container is the parent element of all flex items. The flex container is the only element that has the flex-direction. All direct child elements of a flex container are flex items.

To designate an element as a flex container, set the element’s display property to flex or inline-flex.

.container{ display: flex; }

Once an item is a flex container, there are many properties we can use to specify how its children behave. Some of the properties are:

  • justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly
  • align-items: stretch | flex-start | flex-end | center | baseline
  • flex-direction: row | row-reverse | column | column-reverse
  • flex-wrap: nowrap | wrap | wrap-reverse
  • align-content: stretch | flex-start | flex-end | center | space-between | space-around
  • align-content
  • gap, row-gap, column-gap: 0 this helps to create space between items, without using margin
Flex Items-Properties for the Children

The following are some of the properties meant to be set on the flex item.

  • align-self: auto | stretch | flex-start | flex-end | center | | baseline
  • flex-grow: 0 | 1 Note: 0 means no, 1+ means yes
  • flex-shrink: 0 | 1 Note: 0 means no, 1+ means yes
  • flex-basis: auto | length
  • flex: 0 1 auto | int int len | shorthand for flex-grow, -shrink, -basis
CSS Grid: What is Grid?

The CSS Grid is is a two-dimensional grid-based layout system. It is a tool for arranging items in a grid and helps to lay out items in a two-dimensional grid. The grid can be used to layout entire web pages. Whereas Flexbox is mainly useful for positioning items in a one-dimensional layout. CSS grid is used for two-dimensional layouts. However, CSS Grid is not meant to replace flexbox. The CSS Grid provides many tools for aligning and moving elements across both rows and columns.

the following are important terminologies in CSS Grid:

  • Grid Container
  • Grid Line
  • Grid Track
  • Grid Area
  • Grid Item
  • Grid Cell
Grid Container

Grid Container defines the element as a grid container and establishes a new grid formatting context for its contents. The element on which display: grid is applied. It’s the direct parent of all the grid items. In order to make the grid container you need to set the display property to either grid or inline-grid.

The following are few of the properties meant to be set on the properties for the Parent.

  • display: grid | inline-grid
  • grid-template-columns: track-size | line-name
  • grid-template-areas: grid-area-name | . a period signifies an empty grid cell | none
  • grid-template: none | grid-template-rows | grid-template-columns
  • etc
.container{ display: grid; } .container { grid-template-columns: 100px 50px 100px; grid-template-rows: 80px auto 80px; gap: 15px 10px; }
Grid Items

Grid Items

Grid Items are the children (i.e. direct descendants) of the grid container. A grid container contains grid items. By default, a container has one grid item for each column, in each row, but we can still style the grid items so that they will span multiple columns and/or rows.

Note: These properties have no effect on a grid item: float, display: inline-block, display: table-cell, vertical-align and column-span.

The following are some of the properties meant to be set on the Properties for the Children.

  • grid-column-start: line | span | auto
  • grid-column-end: line | span | auto
  • grid-row-start: line | span | auto
  • grid-row-end: line | span | auto
  • grid-column: start-line | end-line
  • grid-row: start-line | end-line
.item { grid-column-start: | | span | span | auto; grid-column-end: | | span | span | auto; grid-row-start: | | span | span | auto; grid-row-end: | | span | span | auto; }
